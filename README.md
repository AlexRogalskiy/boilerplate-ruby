# Ruby Simple Boilerplate

> To add your credentials, replace the values for `SPACE_ID` and `ACCESS_TOKEN` in _script.rb_.

## On macOS & Linux

### Install & Run

```bash
./run
```

## On Windows

### Requirements

* Ruby 1.9+ (https://rubyinstaller.org/)
* Bundler (https://bundler.io/)

### Install & Run

```
bundle install
ruby script.rb
```
